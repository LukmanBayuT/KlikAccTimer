This is an app to finish the assignment given by KlikAcc Timer

## Installation

//if you have the APK just install it, if you don't contact me!

```bash
pip install foobar
```

## Usage

```python
import foobar

# returns 'words'
foobar.pluralize('word')

# returns 'geese'
foobar.pluralize('goose')

# returns 'phenomenon'
foobar.singularize('phenomena')
```
